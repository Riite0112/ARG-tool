# Chrome Web Store 画像アセット

Chrome Web Store Developer Dashboard の「画像アセット」にアップロードするためのファイルです。

| 用途 | ファイル | サイズ | 備考 |
| --- | --- | --- | --- |
| ショップアイコン | `shop-icon-128.png` | 128x128 | 透明余白付きPNG |
| スクリーンショット | `screenshot-01-investigation-ui-1280x800.png` | 1280x800 | 左パネルと下キーワードバーの基本UI |
| スクリーンショット | `screenshot-02-keyword-copy-1280x800.png` | 1280x800 | キーワード保存とクリックコピー |
| スクリーンショット | `screenshot-03-multi-arg-log-1280x800.png` | 1280x800 | ページ保存ログと進捗管理 |
| プロモーション タイル（小） | `promo-small-440x280.png` | 440x280 | ストア掲載用の小タイル |
| マーキー プロモーション タイル | `promo-marquee-1400x560.png` | 1400x560 | ストア掲載用の大型タイル |

スクリーンショット、プロモーション タイル、マーキー プロモーション タイルは、Chrome Web Store の指定に合わせてアルファなしPNGで作成しています。
