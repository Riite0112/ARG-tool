# ARG探索ツール

ARGやWeb謎解きを遊ぶときの調査を、ページ上に重ねて表示できるChrome/Edge向け拡張機能です。

拡張機能ボタンを押すと、表示メニューからページ一覧とキーワードバーを個別にON/OFFできます。ページ本体には余白を作るので、UIを重ねて隠すのではなく、ゲーム画面を少し狭くして横に並べる形で使えます。

## できること

- 右上の拡張機能ボタンを押すと、ページ一覧とキーワードバーを個別に表示できるメニューを開きます。
- 表示メニューで「ページ」と「キーワード」を両方選ぶと、左パネルと下バーを同時に表示します。
- 表示メニューには、開発者へフィードバックを送るフォーム導線を準備しています。GoogleフォームURLは `src/config.js` に後から設定できます。
- 表示メニュー下に、今後のための「開発者を応援」準備枠を置いています。
- 表示メニューからテーマを選択できます。未到達ページの赤色、再訪ページのオレンジ色と被らない緑・青・紫・灰のアクセントを用意しています。
- 表示メニューの `COPY` をONにすると、ページ上でコピーした文字をキーワードへ自動で一時保存できます。
- 一度開いたARGサイトでは、同じサイト内のページを開いたときに自動で左パネルと下バーを表示します。
- 非表示にしたサイトは自動表示を止め、拡張機能メニューから再表示できます。
- 複数のARGを別々の調査セッションとして保存し、現在のサイトに合うARGへ自動で切り替えます。
- 左上の `+ ARG` ボタンで、現在のサイトを新しいARGとして追加できます。
- 左上のセレクトでARGを手動切り替えでき、タイトル入力欄でARG名を保存できます。
- 左上の削除ボタンで現在のARGを削除でき、左のページ一覧から登録ページを個別に削除できます。
- `TARGET #` にARG全体の想定ページ数を入力できます。
- `CURRENT` に今見ているページ番号を手動入力できます。
- `SUB` に枝番を入れると、`1-1`、`1-2` のように同じページ内のサブページを分けて保存できます。空欄なら今まで通りページ番号だけで保存します。
- `KEYWORD` にそのページへ到達したキーワードを入力できます。
- 保存時に現在のURLを自動取得し、ページ番号、到達キーワード、リンク名、URLをまとめて保存します。
- 同じページ番号と枝番、または同じURLと枝番でもう一度保存した場合は重複させず、既存項目を更新します。
- 登録済みページ番号に欠番がある場合は、左パネルに赤文字で未到達ページを自動表示します。未到達ページは進捗数には含めません。
- パスワード入力などであとから再訪したいページは、左一覧の `!` ボタンでオレンジ表示に切り替えられます。もう一度押すと通常色に戻ります。
- 右上の拡張機能メニュー内のタイマーで攻略時間を計測できます。タイマー開始後にページ保存すると、その時点の経過時間がページ一覧に記録されます。
- 登録済みページを開いたときは、左のページ一覧で該当ページが中央に来るよう自動スクロールします。
- 検索に使いそうなキーワードだけを「一時保存」できます。
- 下バーに保存したキーワードは、クリックするだけでクリップボードへコピーできます。
- 下バーのキーワードは、右クリックまたは `×` ボタンで削除できます。
- キーワードが増えた場合は下バー内だけをスクロールでき、少なくとも2行分のキーワードを見ながら使えます。
- 拡張機能メニュー内の「文字化け」で、文字化けした文字列の復元候補を表示できます。
- 拡張機能メニュー内の「QR読取」で、画像ファイルまたはクリップボード画像のQRコードを読み取れます。
- ページ右下に固定表示される `FILE: 01 / 46` のようなページ表示は、下バーに隠れないよう自動で上へ退避します。
- ページ左下の履歴ボタンや左上の固定ヘッダーも、左パネルと下バーに隠れない位置へ自動で退避します。
- メッセージ入力欄がある画面いっぱいのチャット型ARGは、送信欄が下バーに隠れないようアプリ全体の高さを自動調整します。
- 下バー右側の `?` ボタンから、拡張機能内の説明書を開けます。
- Webページ上で選択したテキストを、右クリックから拡張へ送れます。
- 右クリックから送った選択テキストを、そのままキーワードとして保存できます。
- テキストを左パネルまたは下バーへドラッグ&ドロップして、キーワード欄へ入れられます。
- 保存データはブラウザ内にローカル保存されます。

## ローカルで読み込む

1. Chromeなら `chrome://extensions`、Edgeなら `edge://extensions` を開きます。
2. 右上の「デベロッパー モード」をオンにします。
3. 「パッケージ化されていない拡張機能を読み込む」を押します。
4. このフォルダを選びます: `C:\Users\fire1\Documents\ARG tool`
5. ツールバーの拡張機能アイコンを押すか、`Ctrl+Shift+Y` で表示メニューを開きます。

## 使い方の流れ

1. ARGや謎解きページを開きます。
2. ツールバー右上の拡張機能ボタンを押し、メニューから「ページ」または「キーワード」を選びます。両方選ぶと左パネルと下バーが同時に表示されます。
3. 必要に応じて、拡張機能メニューの `THEME` から見やすいテーマを選びます。
4. そのサイトのURL本体が自動表示対象として記録されます。たとえば `https://kurono-zoo.com/pages/access.html` で開くと、`https://kurono-zoo.com` 配下のページで自動表示されます。
5. 別のARGを始めるときは、そのサイトで左上の `+ ARG` を押します。
6. タイトル入力欄にARG名を入れます。
7. 以前のARGへ戻りたいときは、左上のセレクトから手動で切り替えます。
8. 不要なARGは左上の「削除」、不要なページは左一覧の `×` で削除します。
9. `TARGET #` に総ページ数を入れます。
10. 攻略時間を記録したい場合は、右上の拡張機能メニューを開き、`TIMER` で `Start` を押します。
11. コピーした文字を候補キーワードへ入れたい場合は、拡張機能メニューの `COPY` をONにします。OFFにすると自動保存を止められます。
12. 今見ているページが何ページ目かを `CURRENT` に入れます。
13. 同じページ内のサブページとして分けたい場合は、`SUB` に枝番を入れます。空欄なら通常ページとして保存します。
14. そのページへ到達したキーワードを `KEYWORD` に入れて「ページ保存」を押します。
15. URLは自動取得され、左側の一覧にページ番号、枝番、キーワード、保存時点のタイマー時間が追加されます。
16. まだページ登録しない候補語は、`KEYWORD` に入れて「一時保存」を押します。
17. 下バーに並んだキーワードは、押すだけでコピーできます。
18. 不要なキーワードは、右クリックまたは `×` で削除できます。
19. 文字化けを直したいときは、拡張機能メニューの「文字化け」に貼り付けて「復元」を押します。
20. QRコードを読みたいときは、拡張機能メニューの「QR読取」から画像を選びます。
21. 使い方を確認したいときは、下バー右側の `?` を押します。
22. 画面を広く使いたいときは、拡張機能メニューの「非表示」を押します。

## データ保存

データは `chrome.storage.local` に保存されます。外部サーバーには送信しません。

自動表示対象のサイトURLと複数ARGの調査セッションも同じ保存データ内に保持されます。

別のPCや別ブラウザへ移したい場合のために、JSON書き出し/読み込みUIは今後追加予定です。

## 権限について

- `tabs`: 現在開いているページのタイトルとURLをログに入れるために使います。
- `activeTab`: 選択中のページからテキストを取得するために使います。
- `contextMenus`: 右クリックメニューから選択テキストを送るために使います。
- `scripting`: 既に開いているタブへオーバーレイを表示するために使います。
- `storage`: 調査ログとキーワードをローカル保存するために使います。

## ファイル構成

- `manifest.json`: 拡張機能の設定
- `src/background.js`: 右クリックメニュー、ページ保存、オーバーレイ起動
- `src/config.js`: フィードバックフォームURLなど、公開前に差し替える設定
- `src/content.js`: Webページ上に表示する左パネル/下バーの処理
- `popup.html`, `src/popup.js`, `src/popup.css`: 拡張機能ボタンから開く表示メニュー
- `icons/`: 拡張機能アイコン
