# ARG探索ツール

ARGやWeb謎解きを遊ぶときの調査を、ページ上に重ねて表示できるChrome/Edge向け拡張機能です。

拡張機能ボタンを押すだけで、現在開いているページのリンク名とURLが保存され、Webページ左側の細いパネルに表示されます。画面下側にはキーワード保存バーが出るので、ページを見ながら気になる語句、暗号、合言葉候補をその場で管理できます。

## できること

- 右上の拡張機能ボタンを押すと、現在タブのタイトルとURLを保存できます。
- 拡張機能ボタンから追加したページだけを、Webページ左側のオーバーレイ一覧に表示します。
- 追加ページにはページ番号、リンク名、URLを残せます。
- 同じURLをもう一度追加した場合は重複させず、既存項目を更新します。
- 画面下側のキーワードバーで「主力」と「保留」の保存先を使い分けられます。
- Webページ上で選択したテキストを、右クリックから拡張へ送れます。
- 右クリックから送った選択テキストを、そのままキーワードとして保存できます。
- テキストを左パネルまたは下バーへドラッグ&ドロップして、キーワードとして保存できます。
- 保存データはブラウザ内にローカル保存されます。

## ローカルで読み込む

1. Chromeなら `chrome://extensions`、Edgeなら `edge://extensions` を開きます。
2. 右上の「デベロッパー モード」をオンにします。
3. 「パッケージ化されていない拡張機能を読み込む」を押します。
4. このフォルダを選びます: `C:\Users\fire1\Documents\ARG tool`
5. ツールバーの拡張機能アイコンを押すか、`Alt+Shift+A` で現在ページを保存し、ページ上にオーバーレイを表示します。

## 使い方の流れ

1. ARGや謎解きページを開きます。
2. ツールバー右上の拡張機能ボタンを押し、現在のページを左側の一覧に残します。
3. 気になる文字列はページ上で選択して右クリックし、「選択したテキストをARG探索ツールへ送る」を使います。
4. キーワード候補は画面下側の「KEYWORDS」欄で「主力」または「保留」に保存します。
5. 追加済みページは左側、キーワードは下側に残るので、ページを見ながら探索を続けます。

## データ保存

データは `chrome.storage.local` に保存されます。外部サーバーには送信しません。

別のPCや別ブラウザへ移したい場合のために、JSON書き出し/読み込みUIは今後追加予定です。

## 権限について

- `tabs`: 現在開いているページのタイトルとURLをログに入れるために使います。
- `activeTab`: 選択中のページからテキストを取得するために使います。
- `contextMenus`: 右クリックメニューから選択テキストを送るために使います。
- `scripting`: 既に開いているタブへオーバーレイを表示するために使います。
- `storage`: 調査ログとキーワードをローカル保存するために使います。

## ファイル構成

- `manifest.json`: 拡張機能の設定
- `src/background.js`: 右クリックメニュー、ページ保存、オーバーレイ起動
- `src/content.js`: Webページ上に表示する左パネル/下バーの処理
- `icons/`: 拡張機能アイコン
