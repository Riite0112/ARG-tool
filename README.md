# ARG探索ツール

ARGやWeb謎解きを遊ぶときの調査を、ブラウザのサイドパネルでまとめて扱うChrome/Edge向け拡張機能です。

参照サイトを開きながら、見つけたURL、ページ番号、手がかり、仮説、キーワード、暗号らしき文字列を同じ場所に残せます。データはローカル保存なので、個人用の探索メモとしてすぐ使えます。

## できること

- 開いているタブのタイトルとURLを、ワンクリックで探索ログに追加できます。
- 手動ログにはページ番号、状態、色、URL、メモを付けられます。
- ページ番号に抜けがあると警告します。
- 「主力」と「保留」のキーワード置き場を使い分けられます。
- Webページ上で選択したテキストを、右クリックから拡張へ送れます。
- 選択中の文字列をデコーダへ取り込めます。
- Base64、URLエンコード、シーザー/ROT13、Atbash、モールス、バイナリ、16進数、逆順、URL/数字/大文字の抽出を試せます。
- 調査データ全体をJSONで書き出し、あとから読み込めます。

## ローカルで読み込む

1. Chromeなら `chrome://extensions`、Edgeなら `edge://extensions` を開きます。
2. 右上の「デベロッパー モード」をオンにします。
3. 「パッケージ化されていない拡張機能を読み込む」を押します。
4. このフォルダを選びます: `C:\Users\fire1\Documents\ARG tool`
5. ツールバーの拡張機能アイコンを押すか、`Alt+Shift+A` でサイドパネルを開きます。

## 使い方の流れ

1. ARGや謎解きページを開きます。
2. サイドパネルで「ページ追加」を押し、現在のページをログに残します。
3. 気になる文字列はページ上で選択して右クリックし、「選択したテキストをARG探索ツールへ送る」を使います。
4. キーワード候補は「主力」または「保留」に置いて、必要になったらログ化します。
5. 暗号っぽい文字列は「デコード」タブに貼り付けて、複数の変換結果をまとめて確認します。
6. 区切りのよいところで「JSONを書き出し」からバックアップします。

## データ保存

データは `chrome.storage.local` に保存されます。外部サーバーには送信しません。

別のPCや別ブラウザへ移したい場合は、「データ」タブのJSON書き出し/読み込みを使ってください。

## 権限について

- `tabs`: 現在開いているページのタイトルとURLをログに入れるために使います。
- `activeTab`: 選択中のページからテキストを取得するために使います。
- `contextMenus`: 右クリックメニューから選択テキストを送るために使います。
- `sidePanel`: サイドパネルでツールを表示するために使います。
- `storage`: 調査ログとキーワードをローカル保存するために使います。

## 今後の拡張案

- チームで同じ調査ログを共有するルーム機能
- スクリーンショットや画像メモの添付

- OBSや配信用の読み取り専用ビュー

## ファイル構成

- `manifest.json`: 拡張機能の設定
- `src/background.js`: 右クリックメニューとサイドパネル起動
- `src/content.js`: Webページ上の選択テキスト取得
- `src/panel.html`: サイドパネルの画面
- `src/panel.css`: サイドパネルの見た目
- `src/panel.js`: ログ、キーワード、デコード、JSON入出力の処理
- `icons/`: 拡張機能アイコン
