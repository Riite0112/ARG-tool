# ARG探索ツール プライバシーポリシー

最終更新日: 2026-06-20

ARG探索ツールは、ARGやWeb謎解きの調査ログをブラウザ内で管理するためのChrome/Edge向け拡張機能です。

## 収集・保存する情報

この拡張機能は、ユーザーが機能を利用した場合に、以下の情報をブラウザ内に保存します。

- ARGセッション名
- ユーザーが保存したページ番号
- ユーザーが保存したページURL
- ユーザーが保存したページタイトル
- ユーザーが入力した到達キーワード
- ユーザーが一時保存した候補キーワード
- 自動表示対象のサイトURL
- 非表示にしたサイトURL
- 右クリックメニューからユーザーが明示的に送信した選択テキスト

## 利用目的

保存した情報は、以下の目的にのみ利用します。

- ARGやWeb謎解きの調査ログを表示するため
- ページ番号、URL、到達キーワード、候補キーワードを整理するため
- 保存済みARGサイトで調査UIを自動表示するため
- ユーザーが保存したキーワードをコピーしやすくするため

## 保存場所

情報は `chrome.storage.local` に保存されます。これはユーザーのブラウザ内のローカル保存領域です。

## 外部送信

この拡張機能は、保存した情報を外部サーバーへ送信しません。

この拡張機能は、広告、アクセス解析、トラッキング、リモートコードの実行を行いません。

## 第三者提供

この拡張機能は、保存した情報を第三者へ販売、共有、提供しません。

## クリップボード

ユーザーがキーワードをクリックした場合、そのキーワードをクリップボードへ書き込みます。クリップボードの読み取りは行いません。

## データの削除

ユーザーは、拡張機能内の削除操作により、保存したARG、ページ、キーワードを削除できます。

また、ブラウザの拡張機能削除、またはブラウザ設定から拡張機能の保存データを削除することでも、保存情報を削除できます。

## Chrome Web Store User Data Policy

The use of information received from Google APIs will adhere to the Chrome Web Store User Data Policy, including the Limited Use requirements.
